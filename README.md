
# AppealaRx

An insurance appeal letter generator built for small community health clinics.

## The Problem

When insurance companies deny claims, most small clinics don't have legal staff 
to fight back. Appeals get dropped. Revenue gets lost. Patients lose access to care.

## The Solution

AppealaRx lets clinic staff enter basic claim details and generate a professional, 
ready-to-send appeal letter in seconds — no lawyer needed.

## Who It's For

Small and nonprofit community health clinics handling insurance claim denials.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- AI: Claude API (Anthropic)

## Status

In active development. MVP in progress.
